# Raw data 
Large raw sequence files, alignments, phylogenetic input files, BEAST XML files, BEAST log files, tree files, and state files are not stored in this GitHub repository because of file size.

Raw data used in this project include _R. solani_ sequence datasets downloaded from public databases (NCBI) and/or sample collection. The source and full metadata can be found in (table).
- ITS FASTA files
- TEF-1α FASTA files
- unprocessed alignment files
- trimmed alignment files
- raw phylogenetic input files
